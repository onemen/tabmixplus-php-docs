Mouse - Mouse Clicking
Choose command to perform when clicking on the tab or tabbar - Choose a click action, then specify the action from the list to be applied for the tab and tabbar
Select which action is to be applied for each of the five mouse click events
  • Double-click
  • Middle-click
  • Ctrl-click
  • Shift-click
  • Alt-click

You can enable Firefox default or let other extensions get control for spesific mouse click event by unchecking the box next to spesific action.
