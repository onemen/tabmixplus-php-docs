The future of

Tab Mix Plus

Firefox 57 drops support for all legacy add-ons, including Tab Mix Plus. From Firefox 57 onwards, Firefox will not install Tab Mix Plus.

Tab Mix webExtension

I am currently working on converting Tab Mix Plus code base to the new webExtension format.
Tab mix code base holds more than 35,000 line of code in 139 files.

It is currently unclear when Tab Mix Plus as a WebExtension will be ready, it will not be ready before Firefox 57 official release. I am also not sure it will be possible to implement all Tab mix features in the new format.

Anyone who wants to contribute code or assist in any other way can contact me directly via tabmix.onemen@gmail.com.

Using legacy Tab mix plus with Firefox 57+

If you want to keep using Tab Mix Plus with Firefox 57, you need to install modified version of Firefox.
Currently this version is available only for windows users and in English.
Using the modified version will enables all legacy extensions.

To install Firefox by downloading the .zip file and installing manually, follow these steps:

  • Click the firefox-57.0b14.en-US.win32.zip link or the firefox-57.0b14.en-US.win64.zip link to download the .zip file to your machine.

    Important: You will need to install manually new modified version of Firefox when Firefox updates. I will update the links above to the latest available version.

  • Extract the .zip file to a directory such as 'C:\Program Files (x86)\Mozilla Firefox' for 32-bit or 'C:\Program Files\Mozilla Firefox' for 64-bit.
  • To start Firefox, navigate to the directory you extracted Firefox to and double click the firefox.exe icon.
  • Start Firefox and set the preference 'extensions.legacy.enabled' to true in about:config. Restart Firefox.

Using legacy Tab mix plus with Firefox Nightly:

  • Download and install Firefox Nightly from here.
  • Start Firefox and set the preference 'extensions.legacy.enabled' to true in about:config. Restart Firefox.
  • Install latest development version of Tab mix plus from here here.
